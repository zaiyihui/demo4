## 🔍 查看构建状态

如果你遇到构建问题，可以：

1. **清理并重新构建**：
   ```powershell
   dotnet clean
   dotnet build
   ```

2. **检查依赖**：
   ```powershell
   dotnet restore
   dotnet list package
   ```

3. **查看详细错误**：
   ```powershell
   dotnet build --verbosity detailed
   ```
