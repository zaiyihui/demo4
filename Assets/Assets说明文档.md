# Assets 文件夹说明文档

## 1. 文件夹功能概述
**Assets** 文件夹用于存放项目的静态资源文件，包括应用程序图标、图片、字体等资源。该文件夹是项目资源管理的核心位置。

## 2. 包含内容

| 文件/文件夹 | 文件类型 | 说明 | 重要性 |
|------------|---------|------|--------|
| **avalonia-logo.ico** | 图标文件 | Avalonia 框架默认的应用程序图标 | ⭐⭐⭐ |

## 3. 重要文件说明

### avalonia-logo.ico
- **用途**：应用程序的默认图标，在任务栏、窗口标题栏等位置显示
- **格式**：ICO 格式，支持多种尺寸（16x16、32x32、48x48、256x256 等）
- **位置**：[Assets/avalonia-logo.ico](file:///d:/BC/xiangmu/demo4/demo4/Assets/avalonia-logo.ico)
- **建议**：如果需要自定义应用图标，可替换此文件

## 4. 使用规范

### 4.1 文件命名规范
- 使用清晰、描述性的文件名
- 建议使用英文命名，避免中文字符导致的跨平台问题
- 图标文件使用 `.ico` 格式，图片使用 `.png` 或 `.jpg` 格式

### 4.2 文件组织
- 如资源文件增多，建议按类型建立子文件夹（Images/、Icons/、Fonts/ 等）
- 保持资源文件大小适中，避免过大的图片影响应用启动速度
- 重要资源文件应加入版本控制

### 4.3 在代码中引用
在 Avalonia 中引用资源文件时使用相对路径：
```csharp
// 引用图标
Icon = new WindowIcon("avares://电脑伴侣/Assets/avalonia-logo.ico");
```

## 5. 维护建议

- 定期检查是否有未使用的资源文件，及时清理
- 如需更换应用图标，建议先备份原文件
- 注意资源文件的版权许可，避免侵权
- 如添加新的资源类型，及时更新本文档
