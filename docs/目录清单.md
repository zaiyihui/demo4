## 8. 添加新文档指南

如需添加新文档，请按照以下步骤操作：

1. **确定文档类型和位置**
   - 项目级文档：放置在 `docs/` 文件夹
   - 模块级文档：放置在 `docs/模块文档/` 文件夹
   - 文件夹说明：放置在对应文件夹下

2. **编写文档内容**
   - 使用 Markdown 格式
   - 包含文档版本和日期信息
   - 遵循文档命名规范

3. **更新本目录清单**
   - 在相应表格中添加记录
   - 更新文档统计数据
   - 添加到文件位置索引

4. **提交更改**
   - 一起提交文档和目录清单的更新
   - 记录维护历史
