# 设置服务模块文档

---

## 1. 模块概述

### 1.1 功能描述

设置服务（`SettingsService`）负责管理应用程序的所有配置项，包括：

- **界面配置**：布局模式、颜色主题、字体大小、透明度
- **监控配置**：刷新间隔、游戏模式、显示项开关
- **悬浮窗配置**：启用状态、位置、字体大小、颜色
- **启动配置**：开机自启、启动时最小化

### 1.2 设计目标

| 目标 | 说明 |
|------|------|
| **持久化** | 配置自动保存到文件，重启后保持 |
| **默认值** | 所有配置项都有合理的默认值 |
| **完整性** | 提供配置重置功能 |
| **线程安全** | 支持多线程访问 |

## 4. 数据流程

### 4.1 初始化流程

```
SettingsService 构造
    │
    ├─→ 获取 AppData 路径
    │     └─→ %APPDATA%\ComputerCompanion
    │
    ├─→ 创建目录（如不存在）
    │
    ├─→ 检查 settings.json 是否存在
    │     ├─→ 存在 → LoadSettings()
    │     └─→ 不存在 → 创建默认设置并保存
    │
    └─→ 返回 SettingsService 实例
```

### 4.2 设置加载流程

```
LoadSettings()
    │
    ├─→ 读取 settings.json 文件
    │
    ├─→ JSON 反序列化为 Settings 对象
    │
    ├─→ 验证配置项（处理版本兼容）
    │
    └─→ 保存到 _settings 字段
```

### 4.3 设置保存流程

```
SaveSettings()
    │
    ├─→ JSON 序列化为格式化字符串
    │
    ├─→ 写入 settings.json 文件
    │
    └─→ 处理异常（记录日志）
```

## 5. 关键实现细节

### 5.1 配置文件路径

```csharp
var appDataPath = Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData);
var appFolder = Path.Combine(appDataPath, "ComputerCompanion");
Directory.CreateDirectory(appFolder);
_settingsPath = Path.Combine(appFolder, "settings.json");
```

**设计要点**：
- 使用 `Environment.SpecialFolder.ApplicationData` 获取用户配置目录
- 自动创建目录，避免文件写入失败

### 5.2 设置加载

```csharp
public void LoadSettings()
{
    if (File.Exists(_settingsPath))
    {
        try
        {
            var json = File.ReadAllText(_settingsPath);
            _settings = JsonConvert.DeserializeObject<Settings>(json) ?? new Settings();
        }
        catch (Exception ex)
        {
            Console.WriteLine($"加载设置失败: {ex.Message}");
            _settings = new Settings();
        }
    }
    else
    {
        _settings = new Settings();
        SaveSettings();
    }
}
```

**设计要点**：
- 处理文件不存在的情况
- 处理 JSON 解析失败的情况
- 异常时使用默认设置

### 5.3 设置保存

```csharp
public void SaveSettings()
{
    try
    {
        if (_settings != null)
        {
            var json = JsonConvert.SerializeObject(_settings, Formatting.Indented);
            File.WriteAllText(_settingsPath, json);
        }
    }
    catch (Exception ex)
    {
        Console.WriteLine($"保存设置失败: {ex.Message}");
    }
}
```

**设计要点**：
- 使用格式化输出，便于人工阅读
- 异常处理避免程序崩溃
