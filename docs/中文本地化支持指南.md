# 电脑伴侣 - 中文本地化支持指南

## 概述

本文档详细说明了"电脑伴侣"项目的中文本地化支持配置，确保在所有平台上正确显示中文字符和支持中文输入法。

## 中文支持特性

### 1. 编码配置
- 所有源代码文件使用 UTF-8 编码（无 BOM）
- `.editorconfig` 文件强制统一编码和格式化
- 控制台输出使用 UTF-8 编码

### 2. 字体配置
- 主字体：Segoe UI
- 中文字体：Microsoft YaHei UI, Microsoft YaHei, SimHei
- 跨平台字体：Noto Sans CJK SC, Source Han Sans CN
- 回退机制：如果首选字体不可用，自动使用后备字体

### 3. 文化和区域设置
- 默认区域：zh-CN（简体中文）
- 支持区域：zh-CN（简体中文）, zh-TW（繁体中文）, en-US（英文）
- 数字、日期、货币格式使用中文标准

### 4. 输入法支持
- 全平台支持中文输入法
- Windows 系统：微软拼音、搜狗拼音等
- 正确的输入处理和显示

## 核心配置说明

### .editorconfig

强制所有源文件使用 UTF-8 编码和统一的格式化风格：

```ini
[*]
charset = utf-8
end_of_line = crlf
indent_size = 4
indent_style = space
insert_final_newline = true
trim_trailing_whitespace = true
```

### 全局字体样式 (GlobalStyles.axaml)

确保所有 UI 元素使用正确的中文字体：

```xml
<Style Selector="Control">
    <Setter Property="FontFamily" Value="{StaticResource DefaultFontFamily}"/>
</Style>
```

### 应用程序资源 (App.axaml)

定义中文字体堆栈：

```xml
<FontFamily x:Key="ChineseFontFamily">Microsoft YaHei UI, Microsoft YaHei, SimHei, Noto Sans CJK SC, Source Han Sans CN, sans-serif</FontFamily>
```

### 项目配置 (ComputerCompanion.csproj)

设置默认语言和文化信息：

```xml
<NeutralLanguage>zh-CN</NeutralLanguage>
<SatelliteResourceLanguages>zh-CN;zh-TW;en-US</SatelliteResourceLanguages>
```

## 中文字符串资源

所有界面文本集中在 `Localization/Strings.zh-CN.cs` 中管理，包括：

- 应用程序通用文本
- 主窗口文本
- 悬浮窗文本
- 设置窗口文本
- 系统托盘文本
- 状态和错误提示
- 单位和格式字符串

## 使用指南

### 添加新的中文字符串

1. 在 `Strings.zh-CN.cs` 中添加常量
2. 使用 `Strings.YourStringName` 引用
3. 如需多语言支持，可扩展为资源管理器

### 自定义字体

1. 在 `App.axaml` 中添加新的字体资源
2. 在样式中引用新字体
3. 确保字体在所有目标平台上可用

### 测试中文支持

1. 编译项目：`dotnet build`
2. 运行程序测试中文显示
3. 检查各种 UI 元素的中文字体渲染
4. 测试中文输入法
5. 验证特殊字符和标点符号

## 常见问题

### Q: 中文字符显示为乱码或方块？
A: 检查以下几点：
- 文件编码是否为 UTF-8
- 系统是否安装了中文字体
- 字体堆栈配置是否正确

### Q: 中文输入法无法正常使用？
A: 确保：
- Avalonia 的文本输入控件正确配置
- 系统输入法已正确安装和设置
- 焦点管理正确处理

### Q: 如何添加其他语言支持？
A: 可以扩展当前架构：
1. 创建新的语言资源文件（如 Strings.zh-TW.cs）
2. 实现语言切换功能
3. 加载对应的资源文件

### Q: 在 Linux/macOS 上中文显示有问题？
A: 确保：
- 系统安装了 Noto Sans CJK 系列字体
- 字体堆栈配置包含跨平台字体
- 区域设置正确配置

## 性能考虑

- 字体堆栈按优先级排序，首选字体优先尝试
- 避免使用过多字体变体
- 保持字体大小和字重的合理使用
