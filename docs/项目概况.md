# Computer Companion 项目概况文档

---

## 1. 项目背景与核心目标

### 1.1 项目背景
Computer Companion 是一款基于 Avalonia UI 框架开发的跨平台硬件监控桌面应用程序，旨在为用户提供实时的系统硬件状态监控和性能数据展示。

### 1.2 核心目标

- **实时硬件监控**：提供 CPU、GPU、内存、网络、磁盘、电池等硬件的实时状态监控
- **游戏悬浮窗**：提供 NVIDIA 风格的游戏悬浮窗，显示关键性能指标
- **系统托盘集成**：支持最小化到系统托盘，后台持续监控
- **个性化配置**：支持自定义界面布局、颜色主题、显示项等配置
- **跨平台支持**：基于 Avalonia 框架，支持 Windows、macOS、Linux 平台

## 3. 技术架构设计

### 3.1 架构风格
- **MVVM 架构**：采用 Model-View-ViewModel 架构模式
- **模块化设计**：各功能模块解耦，便于独立开发和测试
- **异步通信**：使用事件驱动和异步任务处理数据更新

## 6. 关键业务流程

### 6.1 应用启动流程

```
用户启动应用
    │
    ▼
解析命令行参数
    │
    ├── 有 --overlay 参数 ──→ 启动悬浮窗模式
    │                          │
    │                          ▼
    │                   初始化 OverlayWindow
    │                   连接 IPC 客户端
    │
    └── 无参数 ──→ 启动主程序模式
                           │
                           ▼
                   初始化 MainWindow
                   启动 IPC 服务器
                   初始化系统托盘
                   启动悬浮窗进程（如启用）
```

### 6.2 数据更新流程

```
定时器触发 (默认 1s)
    │
    ▼
HardwareMonitorService.UpdateData()
    │
    ├── UpdateHardwareData()   ← 从 LibreHardwareMonitor 获取数据
    ├── UpdateNetworkData()    ← 从 NetworkInterface 获取数据
    ├── UpdateDiskData()       ← 从 DriveInfo 获取数据
    └── UpdateBatteryData()    ← 从 Win32 API 获取数据
            │
            ▼
        DataUpdated 事件触发
            │
            ▼
    ViewModel 响应事件更新 UI
```

## 7. 核心类与方法说明

### 7.1 HardwareMonitorService

| 方法 | 功能说明 | 参数 | 返回值 |
|------|----------|------|--------|
| `Start(int intervalMs)` | 启动监控服务 | `intervalMs`: 更新间隔(ms) | void |
| `Stop()` | 停止监控服务 | 无 | void |
| `MarkFrame()` | 标记帧用于 FPS 计算 | 无 | void |
| `MeasureLatencyAsync()` | 异步测量网络延迟 | 无 | Task |

### 7.2 SettingsService

| 方法 | 功能说明 | 参数 | 返回值 |
|------|----------|------|--------|
| `GetSettings()` | 获取当前设置 | 无 | Settings |
| `SaveSettings()` | 保存设置到文件 | 无 | void |
| `LoadSettings()` | 从文件加载设置 | 无 | void |
| `ResetToDefaults()` | 重置为默认设置 | 无 | void |

### 7.3 IpcService

| 方法 | 功能说明 | 参数 | 返回值 |
|------|----------|------|--------|
| `StartServerAsync()` | 启动 IPC 服务器 | 无 | Task |
| `ConnectAsync()` | 连接到 IPC 服务器 | 无 | Task |
| `SendMessageAsync(IpcMessage)` | 发送 IPC 消息 | `message`: 消息对象 | Task |
